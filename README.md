# Student Management System (Python)

## Description
A command-line based Student Management System built using Python.
The system allows users to add, update, delete, and view student records,
with persistent storage using CSV files.

## Features
- Add, update, delete, and view student records
- Input validation and error handling
- Save and load data from CSV files
- Menu-driven user interface

## Technologies Used
- Python
- Dictionaries & Functions
- File Handling (CSV)

## How to Run
1. Ensure Python 3 is installed
2. Run the program:
   python student_management.py

## Learning Outcomes
- Python fundamentals and data structures
- Modular programming using functions
- File I/O operations
- Basic system design logic
